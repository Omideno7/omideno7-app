Omid No 7 Church PWA - Daily Messages Version

این نسخه دارای فایل daily-messages.js است.
اپ به صورت خودکار بر اساس روز ماه، پیام همان روز را نشان می‌دهد.
روز 1 تا 30 = پیام 1 تا 30
روز 31 = پیام روز 30

برای تغییر پیام‌های ماه بعد:
فقط فایل daily-messages.js را عوض کنید و در GitHub آپلود کنید.
نام فایل باید دقیقاً همین باشد: daily-messages.js

نحوه آپلود:
1. فایل ZIP را باز کنید.
2. همه فایل‌ها را در GitHub Repository آپلود کنید.
3. Commit changes بزنید.
4. اگر بعداً فقط پیام‌ها را عوض می‌کنید، فقط daily-messages.js را آپلود کنید.
