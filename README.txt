Omid No 7 Church PWA - Version 1

این نسخه اولیه اپ کلیسای امیدنو۷ است.

شامل:
- صفحه خانه
- پیام روزانه
- جلسات آنلاین FreeConferenceCall
- برنامه جلسات: دعای صبح هر روز ساعت ۵، جلسه کلیسا یکشنبه ساعت ۸ شب به وقت کرواسی
- درخواست دعا
- عضویت، هدایا، تماس
- دو زبانه فارسی و انگلیسی
- هماهنگ با رنگ لوگو: آبی تیره و سبز
- قابل نصب به عنوان PWA روی موبایل بعد از آپلود روی HTTPS

روش تست سریع:
فایل index.html را در مرورگر باز کنید. برای تست کامل PWA باید روی هاست HTTPS آپلود شود.

اگر GoDaddy Website Builder اجازه آپلود HTML کامل نمی‌دهد، از GitHub Pages، Netlify، Vercel یا هاست جدا استفاده کنید.

لینک جلسه:
https://join.freeconferencecall.com/omideno7church

برای فعال‌سازی نوتیفیکیشن واقعی در نسخه بعدی باید Firebase Cloud Messaging اضافه شود.
